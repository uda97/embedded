The sensor and actuator we used

ultrasonic sensor

An ultrasonic sensor is an instrument that measures the distance to an object using ultrasonic sound waves. An ultrasonic sensor uses a transducer to send and receive ultrasonic pulses that relay back information about an object’s proximity.

sound sensor

Sound detection sensor works similarly to our Ears, having diaphragm which converts vibration into signals.

LED actuator

The LED is typically used to provide visual feedback to the user.

buzzer actuator

Typical uses of buzzers and beepers include alarm devices, timers,train and confirmation of user input such as a mouse click or keystroke.
